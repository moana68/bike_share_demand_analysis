# Bike-Share Demand and Station Imbalance Analysis

## Project Overview

Bike-share systems need to keep bicycles available where and when riders need them. Because demand changes by station, time of day, and day of week, some stations may become empty while others become full.

This project will analyze historical bike-share trip data to identify demand patterns and recurring station imbalances. The long-term goal is to understand when and where rebalancing is most needed and to explore whether future demand can be predicted.

## Main Questions

* When is bike-share demand highest?
* Which stations have the highest number of trip starts and trip ends?
* Which stations tend to lose or gain the most bicycles?
* How do weekday and weekend demand patterns differ?
* Which station imbalances occur repeatedly?
* Can future station demand be predicted using historical data?
